# Bana Väg – Phase 1 Tools

These standalone web tools support early charitable coordination without institutional commitment.

## How to Use
1. Open index.html in any browser
2. Choose the tool that fits your current need
3. Data is saved in your browser (localStorage) — private and portable
4. Share the link with partners — no account needed

## Current Tools
- Food Rescue Board — daily surplus coordination
- Skill & Service Registry — voluntary professional help
- Seasonal Needs Coordinator — gift and clothing drives
- Grant Navigator — foundation application tracking

All tools respect the Second Coat Principle: only surplus is coordinated, primary obligations remain untouched.

For theological foundation, see the white papers.
